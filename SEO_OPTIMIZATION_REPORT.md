## 1. Анализ текущего состояния сайта (Технический аудит)

Сайт `sibdezomsk.ru` имеет хорошую базовую структуру, использует HTML5 и имеет адаптивный дизайн. Однако, была выявлена **критическая техническая проблема**, которая могла быть основной причиной плохого ранжирования.

### 1.1. Выполненные технические исправления

Основная проблема заключалась в **несоответствии абсолютных URL-адресов** в мета-тегах и микроразметке. Сайт, размещенный на GitHub Pages, использовал ссылки на домен `incrouble18.github.io/sibdez/` вместо основного домена `sibdezomsk.ru`. Это приводило к **дублированию контента** и некорректной передаче веса страницы, что критически важно для SEO.

**Внесенные изменения:**

| Элемент | Проблема | Решение |
| :--- | :--- | :--- |
| **Canonical URL** | Указывал на GitHub Pages. | Заменен на `https://sibdezomsk.ru/` во всех HTML-файлах. |
| **Open Graph URL** | Указывал на GitHub Pages. | Заменен на `https://sibdezomsk.ru/` во всех HTML-файлах. |
| **Schema.org URL** | Указывал на GitHub Pages. | Обновлен на `https://sibdezomsk.ru/` в микроразметке. |
| **Sitemap.xml** | Некорректный URL главной страницы. | Исправлен на `https://sibdezomsk.ru/`. |

**Результат:** Устранение этой технической ошибки является **первым и самым важным шагом** для улучшения индексации и ранжирования. Поисковые системы теперь будут корректно воспринимать `sibdezomsk.ru` как основной ресурс.

### 2.2. Рекомендации по контенту и структуре

Для вывода сайта на 1-е место в Омске, необходимо реализовать следующую стратегию:

1.  **Расширение контента на страницах услуг:**
    *   Увеличить объем текста на `klopy.html`, `tarakany.html`, `gryzuny.html`, `blohi.html`.
    *   Включить подробные разделы: "Как подготовиться к обработке", "Что делать после обработки", "Часто задаваемые вопросы (FAQ)".
    *   Оптимизировать заголовки H2/H3 для лучшей структуры и охвата ключевых слов.
2.  **Создание новых целевых страниц:**
    *   **Дезинфекция:** Создать отдельную страницу для услуги дезинфекции (от вирусов, плесени, запахов), так как это высокочастотный запрос.
    *   **Обработка от клещей:** Создать отдельную страницу для акарицидной обработки участков.
3.  **Усиление E-A-T:**
    *   Создать страницу **"О компании"** с фотографиями сотрудников, оборудования и подробной историей.
    *   Разместить сканы **сертификатов и лицензий** на отдельной странице и сослаться на них с главной.
